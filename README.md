## Description

Repository made with purpose of learning backend web aplications development. Backend of https://github.com/SamuelOUS/StayNest

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
